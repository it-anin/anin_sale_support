# 🚀 คู่มือเริ่มต้นใช้งานด่วน

## เริ่มใช้งานใน 3 ขั้นตอน

### 1️⃣ ติดตั้ง (ครั้งแรกเท่านั้น)

```bash
npm install
```

รอประมาณ 2-3 นาที

### 2️⃣ เริ่มต้นใช้งาน

```bash
npm run dev
```

โปรแกรมจะเปิดอัตโนมัติที่: **http://localhost:3000**

### 3️⃣ ใช้งาน

1. คลิก **"📂 อัพโหลด CSV"**
2. เลือกไฟล์จาก POS
3. เลือกสินค้าที่ต้องการพิมพ์
4. คลิก **"🖨️ พิมพ์ป้ายราคา"**

---

## 📋 ไฟล์ CSV ต้องมีคอลัมน์

- **A**: Barcode (เลขบาร์โค้ด)
- **E**: SKU (รหัสสินค้า)
- **F**: Name (ชื่อสินค้า)
- **G**: Unit (หน่วย)
- **I**: Price (ราคา)

## 🖨️ การพิมพ์

### ตั้งค่าเครื่องพิมพ์:
- ✅ กระดาษ: A4
- ✅ Margins: ปกติ
- ✅ Scale: 100%

### ขนาดป้าย:
- กว้าง: 4.5 cm
- สูง: 4.0 cm
- จำนวน: 28 ป้าย/หน้า

---

## ❓ แก้ปัญหาเบื้องต้น

### ไม่เจอคำสั่ง npm
ต้องติดตั้ง Node.js ก่อน: https://nodejs.org

### พอร์ต 3000 ถูกใช้งาน
แก้ไขใน `vite.config.ts`:
```ts
server: {
  port: 3001  // เปลี่ยนเป็นพอร์ตอื่น
}
```

### บาร์โค้ดไม่แสดง
- ตรวจสอบว่าบาร์โค้ดเป็นตัวเลข
- ต้องมี 8-13 หลัก
